What problem does rofi-notes-org solve?

It helps you write down your thoughts/idas/todos while programming just in the matter of hitting a key, thus freeing your mind from overwhelming thoughts and distractions while at the same time keeping you in the zone.

It should also be considered a great component of your GTD system.

How does it solve it?

Creating and accessing a sticky note should be very, very simple, up to the point that such a common operation doesn't even need its own application. rofi-notes-org uses Rofi to gather input from the user and Emacs Org-mode format to store its data in a plain text file. It's not a standalone application but rather an extension to Rofi.

To learn more and look at the source code, visit its Github repo.
