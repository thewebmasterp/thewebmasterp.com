    25 + 5 Clock 🍅

    Live Github

    Created on October 12, 2020 with React.js and SCSS, this beautiful yet functional Pomodoro timer allows you to also configure it manually. This is a standard FCC project of course but mine is special because it is really, really well-looking 😎. It's blue when in "Session" mode and orange (for fun) in "Break" mode + a nice transition between the two (look at the gif 😍).

    Drum Machine 🥁

    Live Github

    Created on September 25, 2020 with React.js and SCSS, this drum machine allows you to play sweet sounds either by pressing keys on your keyboard or by clicking the buttons on the colorful claviature with your mouse 🎶🎶. This is a standard FCC project of course but mine is special because I made it so that colors are changing with each click of a button, thus adding more satisfaction to your game 🌈.
